# Lab 5 (меню + три задания)

Запуск:
```
cd labs/lab5/src/main/java
javac -encoding UTF-8 lab5/*.java
java lab5.Lab5Menu
```

Содержимое:
1) `IntegerSet` — множества из диапазона 0..100: union, intersection, insert, delete, equal, toString.
2) `Rational` — рациональные числа с сокращением дробей, статические операции, вывод в виде `a/b` и в десятичном виде.
3) `TicTacToe` — игра 3x3 для двух игроков (X, O, EMPTY). Проверка победы и ничьей; демо через консоль.
