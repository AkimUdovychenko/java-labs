# Lab 3 (меню + три задания)

Запуск:
```
cd labs/lab3/src/main/java
javac -encoding UTF-8 lab3/*.java
java lab3.Lab3Menu
```

Содержимое:
1) `FactorialTable` — факториалы 1..20 в `long`, таблица, комментарии про 100! (нужен BigInteger).
2) `RetailerSwitch` — магазин: ввод пар <номер, количество>, `switch` по цене, цикл с сентинелом (0).
3) `PiLeibniz` — ряд Лейбница для π: таблица значений (первые 20 и каждые 10000 до 200000), а также k, на котором впервые получается 3.14159.
