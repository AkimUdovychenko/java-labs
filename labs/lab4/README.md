# Lab 4 (меню + три задания)

Запуск:
```
cd labs/lab4/src/main/java
javac -encoding UTF-8 lab4/*.java
java lab4.Lab4Menu
```

Содержимое:
1) `GuessNumber` — угадай число 1..1000 с подсказками, возможность играть снова.
2) `DiceStats` — 36,000,000 бросков двух кубиков: частоты сумм 2..12, таблица.
3) `AirlineSeating` — бронирование 10 мест: бизнес 1-5, эконом 6-10, массив boolean, предложение альтернативы.
